UpCASE
| Upload | Correct | Annotate | Search | Export |

UpCASE is an open source web application for building and maintaining language resources. It provides a wide range of options for viewing, evaluating, manipulating and enriching text collections. UpCASE is conceived as a multifunctional webapp where users can upload text documents (plain, pdf or scanned and ocr'ed). While being able to search over the uploaded text, users can correct, annotate and export it into different formats.

UpCASE comprises the experiences of years of work on Romansh texts and lexical resources. In several earlier projects we have developed web-based editing tools for collaborative OCR correction and annotation and a framework for collaborative lexicon creation and maintenance. UpCASE combines the key features of these projects, namely collaborative corpus construction, enrichment and maintenance (for more info on previous projects please check the link list below).

UpCASE is continuously being developed on GitHub. Under the following link you can try out a prototype as applied to the Romansh Chrestomathy (RC) compiled by Caspar Decurtins. The RC is meant to be the starting point of Biblioteca Digitala, a collection of Romansh texts.
